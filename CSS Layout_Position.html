THe position Property

The position propety specifies the type of positioning method used for an element.

There are five type different position values:

1.Static
2.relative
3.fixed
4.absolute
5.sticky

Position: Static;

An element with position: static; is not positioned in any special way; it is alwyas positioned according to the normal flow for the page.

This div position is static.

Position: Relative;

An element with position: Relative; is positioned relative to its normal position.

Note:

Other content will not be adjusted fit into any gap left by the element.

This div position is relative.

Position:fixed;

An element with position: fixed; is positioned relative to the viewport, which means it always stays in the same place even if the page is scrolled.

Note:

Notice the fixed element in the lower-right corner of the page.

This div position is fixed.

Position: absolute;

An element with position: absolute; is positioned relative to the nearest positioned ancestor (instead of positioned relative to the viewport, like fixed).

Note:
However; if an absolute positioned element has no positioned ancestors, it uses the document body, and moves along with page scrolling.

This div element has position relative.
This div positon is absolutes.
